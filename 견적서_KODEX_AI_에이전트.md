# 5. 월 사용자 규모별 LLM 비용 시뮬레이션

FunETF 앱 MAU 중 챗봇 활성 사용률 20%, 활성 사용자 월 평균 25회 질의 기준으로 산정합니다. AI 챗봇 특성상 반복 사용률이 높고, ETF 시장 변동 시 질의가 급증하는 점을 반영하여 여유 있게 산정합니다.

## (1) 시뮬레이션 전제

| 항목 | 수치 |
|------|------|
| MAU 대비 챗봇 활성 사용률 | 20% |
| 활성 사용자 월 질의 수 | 25회 |
| 질의당 입력 토큰 | 500 (사용자) + 2,000 (시스템 프롬프트) + 4,000 (도구 결과) = 6,500 |
| 질의당 출력 토큰 | 1,000 |
| 질의당 총 토큰 | 입력 6,500 + 출력 1,000 = 7,500 |
| 임베딩 토큰 (RAG) | 질의당 500토큰 |
| Claude Sonnet 4.6 입력 단가 | $3 / 1M 토큰 |
| Claude Sonnet 4.6 출력 단가 | $15 / 1M 토큰 |
| Claude Haiku 4.5 (의도분류) | 입력 $1 / 출력 $5 / 1M 토큰 |
| 임베딩 단가 (text-embedding-3-large) | $0.13 / 1M 토큰 |

## (2) 규모별 월간 비용

| 항목 | MAU 10만 | MAU 30만 | MAU 50만 |
|------|---------|---------|---------|
| 챗봇 활성 사용자 (20%) | 2만 명 | 6만 명 | 10만 명 |
| 월 총 질의 수 (×25회) | 50만 회 | 150만 회 | 250만 회 |
| LLM 입력 토큰 | 32.5억 | 97.5억 | 162.5억 |
| LLM 출력 토큰 | 5억 | 15억 | 25억 |
| **Claude Sonnet 비용** | **$9,750 + $7,500 = $17,250** | **$29,250 + $22,500 = $51,750** | **$48,750 + $37,500 = $86,250** |
| Claude Haiku (의도분류) | $150 | $450 | $750 |
| 임베딩 비용 | $33 | $98 | $163 |
| **LLM 월 합계 (USD)** | **$17,433** | **$52,298** | **$87,163** |
| **LLM 월 합계 (원화, 1USD=1,350원)** | **약 2,353만원** | **약 7,060만원** | **약 11,767만원** |

## (3) 비용 최적화 전략

| 전략 | 절감 효과 | 적용 시 예상 비용 (MAU 30만 기준) |
