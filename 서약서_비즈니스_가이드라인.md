# 비즈니스 가이드라인 준수 서약서

---

(주)바틀(이하 "서약자"라 함)은 삼성자산운용 주식회사(이하 "당사"라 함)의 **KODEX ETF AI 에이전트 서비스 구축 프로젝트** 입찰 참여 및 수행과 관련하여, 아래의 비즈니스 가이드라인을 성실히 준수할 것을 서약합니다.

---

## 제1조 (비밀유지 의무)

1. 서약자는 본 입찰 과정에서 취득한 일체의 정보(RFP, 기술 자료, 업무 자료, 내부 시스템 정보, 고객 정보 등)를 외부에 유출하지 아니한다.
2. 서약자는 취득한 정보를 본 프로젝트의 제안서 작성 및 프로젝트 수행 목적 이외에 사용하지 아니한다.
3. 입찰 탈락 또는 프로젝트 종료 시, 취득한 모든 자료를 즉시 반환하거나 완전히 파기하고, 파기 확인서를 제출한다.

## 제2조 (사내데이터 유출 방지)

1. 서약자는 당사의 사내데이터를 외부 퍼블릭망(Public Network)으로 전송하지 아니한다.
2. 모든 데이터 처리 및 전송은 엔터프라이즈급 보안 환경(VPN, 암호화 통신, 접근 제어 등) 하에서 수행한다.
3. 개발 및 테스트 과정에서 사용하는 데이터는 당사가 승인한 보안 환경 내에서만 취급한다.

## 제3조 (관련 법규 및 보안규정 준수)

서약자는 프로젝트 수행 시 다음의 법규 및 당사 보안규정을 엄격히 준수한다.

1. **전자금융거래법** 및 동법 시행령
2. **전자금융감독규정** 및 모범규준
3. **전자금융감독규정 제14조의 2** (클라우드컴퓨팅서비스 이용절차 등)
4. **개인정보보호법** 및 동법 시행령, 시행규칙, 고시
5. **정보통신망 이용촉진 및 정보보호 등에 관한 법률** 및 동법 시행령, 시행규칙, 고시
6. 기타 금융 IT 관련 법규 및 당사가 정하는 내부 보안규정

## 제4조 (MCP 레이어 데이터 접근 제어)

1. MCP(Model Context Protocol) 레이어를 통한 데이터 접근은 당사가 정의한 접근 권한 체계에 따라 엄격히 제어한다.
2. MCP 서버에서 제공하는 도구(Tool) 및 리소스에 대한 접근은 역할 기반 접근 제어(RBAC)를 적용하며, 비인가 접근을 차단한다.
3. MCP 레이어를 통해 조회 또는 처리되는 금융 데이터, ETF 상품 정보, 고객 정보 등은 로깅 및 감사 추적(Audit Trail)이 가능하도록 한다.

## 제5조 (보안관리 체계)

1. **보안관리자 지정**: 프로젝트 착수 시 전담 보안관리자를 지정하고, 당사에 서면 통보한다.
2. **보안관리 계획 수립**: 프로젝트 착수 후 14일 이내에 보안관리 계획서를 수립하여 당사의 승인을 받는다.
3. **자료정보 보안관리**: 프로젝트 관련 일체의 자료 및 정보는 암호화 저장하며, 권한이 부여된 인력만 접근할 수 있도록 관리한다.
4. **물리적 환경 보안**: 프로젝트 수행 장소의 물리적 보안(출입 통제, 반출입 관리, 화면 보안 등)을 확보한다.
5. **투입인력 보안**: 프로젝트에 투입되는 모든 인력에 대해 개별 보안 서약서를 징구하고, 보안 교육을 실시한다. 인력 변경 시 즉시 당사에 통보하며, 퇴출 인력의 접근 권한을 즉시 회수한다.
6. **내/외부망 접근 보안**: 당사 내부망과 클라우드 간 전용선 또는 VPN 기반 보안 터널링을 구성하고, 외부 접근을 원천 차단(VPC/ACL 적용)한다.

## 제6조 (개발 완료 후 데이터 삭제)

1. 프로젝트 개발 완료 및 검수 후, 서약자의 로컬 환경(개발 PC, 테스트 서버, 클라우드 환경 등)에 보관된 당사 관련 데이터를 전량 삭제한다.
2. 삭제 완료 후 **삭제 확인서**를 작성하여 당사에 제출한다.
3. 삭제 대상에는 소스코드 사본, 데이터베이스 덤프, 로그 파일, 테스트 데이터, 문서 사본 등 일체의 자료를 포함한다.

## 제7조 (LLM API 활용 시 데이터 보호)

1. LLM(대규모 언어 모델) API 활용 시, 당사의 데이터가 모델 학습에 활용되지 않도록 **데이터 학습 제외(Zero Data Retention, Opt-out)** 설정을 적용한다.
2. LLM API 호출 시 개인정보 및 민감 정보가 포함되지 않도록 **프라이버시 필터링**(PII 마스킹, 데이터 익명화 등)을 적용한다.
3. LLM API 제공업체와의 계약에 데이터 비학습 조항이 포함되어 있음을 증빙할 수 있도록 한다.

## 제8조 (목적물의 소유권 및 지적재산권)

1. 본 프로젝트를 통해 개발된 일체의 산출물(MCP 서버 소스코드, 설계 문서, 데이터 모델, 학습된 모델, 프롬프트, 설정 파일 등)의 소유권 및 지적재산권은 **삼성자산운용 주식회사에 귀속**된다.
2. 서약자는 당사의 사전 서면 동의 없이 산출물을 복제, 배포, 공개, 타 프로젝트에 활용하지 아니한다.
3. 서약자는 본 계약을 통하여 당사에 제공되거나 이용된 지적재산권(특허, 저작권 등)이 제3자의 지적 재산권을 침해하지 않을 것을 보증한다.

## 제9조 (위반 시 책임)

1. 서약자가 본 서약서의 내용을 위반할 경우, 당사는 **계약 해제 또는 해지**를 할 수 있다.
2. 서약자는 위반으로 인하여 당사에 발생한 모든 직접적·간접적 **손해에 대한 배상 책임**을 부담한다.
3. 서약자는 위반 사실이 확인될 경우, 관련 법령에 따른 민·형사상 책임을 감수한다.
